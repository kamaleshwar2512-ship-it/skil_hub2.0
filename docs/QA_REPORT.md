## 6. Stability Recommendations

1.  **Component-Level Error Boundaries**: Wrap individual `PostCard.jsx` and `ProjectCard.jsx` instances in separate boundaries. If one card fails due to corrupted data, the rest of the feed remains interactive.
2.  **Mobile Navigation (Bottom Nav)**: Implement a fixed bottom navigation bar for mobile to ensure users can reach their notifications and explore views.
3.  **Strict Data Validation**: Add Zod or Joi validation on the Backend controllers to prevent `NULL` values or malformed objects from reaching the database in the first place.
